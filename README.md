The gap between a prototype and a manufacturable board is almost always a process gap, not a design gap. Altium Designer keeps schematic, layout, library, simulation and fabrication output in a single unified environment so nothing falls through the cracks between tools.

### Design flow

```
Schematic capture (hierarchical / flat)
        │
        ▼
Component library (Altium 365 cloud · ActiveBOM)
        │
        ▼
PCB layout (Interactive router · Length tuning · Via stitching)
        │
        ▼
Signal integrity (HyperLynx lite · reflection · crosstalk)
        │
        ▼
3D assembly validation (STEP import · MCAD clearance check)
        │
        ▼
Fabrication output (Gerber X2 · ODB++ · IPC-2581 · BOM · pick-and-place)
```

### Layer capabilities

| Stack level | Altium supports |
|---|---|
| Layer count | Up to 64 copper layers |
| Impedance | Controlled-impedance traces with calculator |
| HDI | Blind / buried vias · Microvias · Back-drill |
| Rigid-flex | Flexible region definition · Bend radius rules |

### Routing tools

Interactive router · ActiveRoute (auto-route selected nets) · Length tuning (meandering) · Differential pair router · Micro-via fan-out · Dynamic copper pour

### Output formats

Gerber 274X / X2 · ODB++ · IPC-2581 · NC Drill (Excellon) · Pick-and-place CSV · 3D STEP · PDF schematic · DXF board outline
